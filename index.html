About Me

Hi! I’m Aatra Sadaf, a passionate Front-End Web Developer who loves transforming ideas into clean, modern, responsive websites. I enjoy building smooth user experiences, stylish UIs, and solving problems through creative coding.

I specialize in crafting fast, visually appealing, and functional web interfaces using HTML, CSS, JavaScript, and modern UI patterns. My focus is always on writing clean code and creating designs that feel fresh and up-to-date.

I started my web development journey by learning the foundation of web technologies and slowly moved toward building real projects like apps, calculators, weather dashboards, and mini-web systems. Every project improves my logic, creativity, and confidence.

My vision is to grow into a full-stack developer and work on meaningful projects where design and logic work together. I love clean UI, responsive layouts, smooth animations, and learning something new every day.
